# 販売用 - 追加機能と利用規約・プライバシーポリシー

## 追加推奨機能
販売前に検討すべき追加機能:

### 1. 管理者機能
- [ ] ユーザー一覧・管理画面
- [ ] サブスク状態の確認・手動変更
- [ ] 統計情報（登録者数、解約率など）

### 2. セキュリティ強化
- [ ] ログイン試行制限（ブルートフォース対策）
- [ ] 2段階認証（オプション）
- [ ] IPアドレス制限（管理画面）
- [ ] セッションタイムアウト

### 3. ユーザー機能
- [ ] プロフィール編集
- [ ] メールアドレス変更
- [ ] アカウント削除（GDPR対応）
- [ ] データエクスポート（GDPR対応）

### 4. 課金関連
- [ ] 複数プラン（月額/年額、機能別）
- [ ] クーポンコード
- [ ] アフィリエイト
- [ ] 請求書発行

## 法務書類テンプレート

### 利用規約
最低限含めるべき内容:
- サービス内容と範囲
- 禁止事項
- 知的財産権
- 免責事項
- 解約・返金ポリシー
- 準拠法と管轄裁判所

### プライバシーポリシー
最低限含めるべき内容:
- 収集する個人情報（メール、決済情報など）
- 利用目的
- 第三者提供（Stripe等）
- Cookie利用
- 問い合わせ先

### 特定商取引法表記（日本）
- 事業者名
- 代表者名
- 所在地
- 連絡先
- 料金
- 支払方法・時期
- 返金条件

## 推奨ツール
- **利用規約作成**: Termlyなどのジェネレーター
- **プライバシーポリシー**: iubenda、Termly
- **GDPR対応**: データ削除・エクスポート機能
- **監視**: Sentry（エラー）、Mixpanel（分析）

## 次のステップ
1. 上記の法務書類を作成（弁護士監修推奨）
2. `/terms` `/privacy` ページを追加
3. サインアップ時に同意チェックボックス
4. セキュリティ監査（脆弱性スキャン）
5. 負荷テスト
6. カスタマーサポート体制
